Перед началом работы необходимо создать папки casebook_logs и main_server.
В 1-ой будет хранится лог общения с сервером casebook, во 2-ой с нашим сервером.

В настройках указываете имя файла с компаниями и настраиваете с какой строки
начать (укажите 0 чтобы начать с начала файла) и сколько строк прочитать
(укажите 0 чтобы прочитать весь файл). Все нужные вам параметры прокоментированы.

Также не забудьте установить все зависимовти из файла requirements.txt.
Код написан под python 3.4.3
